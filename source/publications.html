Variable Density Mass Transfer Correction Rn

 

In the paper A note on the composition-dependency of the density within the mass transfer layer, we derived the equation for the mass transfer correction factor Rn, which accounts for the effect of varying density (ρ) in a binary mixture. The model is particularly relevant for high-flux gas absorption or evaporation where the standard constant-density assumptions fail.

 

 

The ratio of the density at the interface to the density in the bulk is modeled using a linear dependency parameter β and the mass fraction ω1:

 

ρ12 = 1 / (1 − β · ω1)

 

where:
- β: A parameter representing the normalized density difference between components.
- ω1: The mass fraction of component 1.

 

The calculation of Rn relates the actual mass flux to the flux predicted by constant-density theories:

 

Rn = 1 + (1/ρ12 - 1) · [1/ln(1/(1 - ω1)) - (1 - ω1)/ω1]

 

This equation adjusts the mass transfer driving force to account for the "bulk flow" (Stefan flow) induced by the density gradient at the interface.

 

Rn as β and ρ12 vary

 

import numpy as np
omg1=np.linspace(0.00001, 0.99999, 25)
betta=[-1, -0.75, -0.5, -0.25, 0, 0.25, 0.5, 0.75, 1]
for BETTA in betta:
    for OMG1 in omg1:
        RHO12 = 1/(1 - BETTA * OMG1)
        Rn=1 + (1/RHO12 - 1) * (1/np.log(1/(1 - OMG1)) - (1 - OMG1)/OMG1)
        print (BETTA, RHO12, Rn)

Falling Film Mass Transfer & Penetration Theory

 

In the paper Revisiting ‘penetration depth’ in falling film mass transfer, we calculated mass transfer characteristics in falling liquid films. We compared the Infinite Penetration (Higbie's theory) and Finite Penetration models to determine how deep a solute penetrates a moving film under various physical conditions.

 

 

Two python scripts are provided to calculate the mass transfer characteristics in falling liquid films. The scripts are written in Python 3.x and use the NumPy library for numerical calculations. The scripts solve the governing equations for convective mass transfer in a laminar falling film. One analyzes how different diffusion coefficients impact the penetration depth and flux (click here). The other analyzes the effect of film flow rate on mass transfer efficiency (click here).

 

Film Hydrodynamics
The film thickness (δ) and average velocity (u) are calculated based on the film flow rate (Γ) and fluid properties (density ρ, viscosity μ):

 

δ = (3μΓ / ρ²g)1/3

u = Γ / (ρδ)

 

Infinite Penetration Model
For short contact times or thick films, the solute is assumed not to reach the wall. The local mass transfer coefficient (kc) is:

 

kc(y) = (3uD / 2πy)1/2

 

Finite Penetration Model
When the solute reaches the wall (finite depth), the concentration profile is solved using a Fourier series expansion. The code determines a characteristic parameter (ξ) by solving for the root of the error function:

 

Error = ( Σn=0∞ (-1)n⁄2n+1 exp( - [ (2n+1)π⁄2ξ ]2 · Dy⁄u ) ) - π⁄4

 

Mass Flux Calculation (NA)
The local mass flux at the interface is determined by:

 

NA = kc · (cAi - cA0)

 

where cAi is the interfacial concentration and cA0 is the bulk concentration.

 

Usage Instructions
1. Open the desired script.
2. Adjust the Test Data section to match your physical system:

 

L = 1         # Wall length (m)
gamma = 0.05  # Film flow rate (kg/m.s)
cAi = 0.0366  # Interfacial concentration

Thermokinetic Model of Phase Inversion

 

In the paper On the search of rigorous thermo-kinetic model for wet phase inversion technique, we developed a rigorous thermo-kinetic model for the wet phase inversion process, with special emphasis on the role of the diffusion-thermo effect at the interface. The model provides a comprehensive mathematical derivation and mapping of the logic implemented in this phase inversion simulation project.

 

 

System Constants & Parameters
The physical constants are defined in CONSTANTS.m. For the default Water (1) / Acetone (2) / Polymer (3) system:

 

Constant Description Value
ρ01 Density of Water 1.0 g/cm³
ρ02 Density of Acetone 0.7857 g/cm³
ρ03 Density of Polymer 1.31 g/cm³
Mw1 Molecular weight (nonsolvent) 18.0 g/mol
Mw2 Molecular weight (solvent) 58.08 g/mol
Mw3 Molecular weight (polymer) 40,000 g/mol
vi Specific volume of component $i$ $1/\rho_{0i}$
Vi Molar volume of component $i$ $v_i \cdot M_{wi}$

 

Solution Properties (DENSTY.m & PHIs.m)

 

- Mixture Density
The density ρ is calculated using a mole-fraction based mixing rule from the equation of state:

 

rρ = ω1 · ((ρ03 − ρ01) / (ρ01ρ03))
+ ω2 · ((ρ03 − ρ02) / (ρ02ρ03))
+ 1 / ρ03

 

ρ = 1 / (rρ)

 

- Volume Fraction Conversion
The volume fractions φ are converted from mass fractions ω using the following logic:

 

D = α ω1 + β ω2 + γ

 

Where:
- α = V1/Mw1 - V3/Mw3
- β = V2/Mw2 - V3/Mw3
- γ = V3/Mw3

 

Then:

 

φ1 = (V1/Mw1) ω1 / D
φ2 = (V2/Mw2) ω2 / D
φ3 = (V3/Mw3) (1 − ω1 − ω2) / D

 

Thermodynamic Logic (FLORYHOGGINSCOEFF.m & FGcomponents.m)

 

- Flory-Huggins Interaction Parameters
Parameters Gij are concentration-dependent. Let u2 be the solvent volume fraction on a polymer-free basis: u2 = \frac{\phi_2}{\phi_1 + \phi_2}.

 

- Interaction G12 (Nonsolvent-Solvent):
G12 = 0.661 + (0.417 / (1 − 0.755 u2))
∂G12/∂u2 = (0.417 · 0.755) / (1 − 0.755 u2)2

 

- Interaction G23 (Solvent-Polymer):
G23 = 0.535 + 0.11 \phi_3
∂G23/∂\phi3 = 0.11

 

- Interaction G13 (Nonsolvent-Polymer):
G13 = 1.4

 

- Thermodynamic Gains (Q1, Q2, S1, S2)
These terms represent the derivatives of chemical potentials ∂μi/∂φj (excluding the RT factor):

 

Q1 =
1/φ1 − 1 + V13 + φ2(V12G23 − G12) − (φ2 + 2φ3)G13 + (φ1 − 2u2)u22 (∂G12/∂u2) + 3V12φ2φ3 (∂G23/∂φ3) + u1u23 (∂2G12/∂u22) + V12φ2φ32 (∂2G23/∂φ32)

 

Q2 =
−V12 + V13 + (φ2 + φ3)(G12 − G13) + V12(φ2 − φ3)G23 + u1u2(u2 − u1 − φ1) (∂G12/∂u2) + V12φ3(3φ2 − φ3) (∂G23/∂φ3) − (u1u2)2 (∂2G12/∂u22) + V12φ2φ32 (∂2G23/∂φ32)

 

S1 =
−V21 + V23 + (φ1 + φ3)(V21G12 − G23) + V21(φ1 − φ3)G13 + V21u1u2(φ2 + u2 − u1) (∂G12/∂u2) + φ3(3φ2 − 1) (∂G23/∂φ3) − V21(u1u2)2 (∂2G12/∂u22) + φ2φ32 (∂2G23/∂φ32)

 

S2 =
1/φ2 − 1 + V23 + V21φ1(G13 − G12) − (φ1 + 2φ3)G13 + V21u12(2u1 − φ2) (∂G12/∂u2) + φ3(4φ2 + φ1 − 2) (∂G23/∂φ3) + V21u13u2 (∂2G12/∂u22) + φ2φ32 (∂2G23/∂φ32)

 

Continuity & Flux Logic (COEFFICIENTS.m, ABCD.m, BDCA.m)
- Kinetic Mobility
Friction coefficients ξij are calculated from self-diffusion coefficients Di, where D2 is given by the solvent self-diffusion in FRICTIONCOEF.m.
ξ23 = (ρ03 Mw3) / D2*

 

- Mobility Matrix Elements (ABCD.m)
The elements A, B, C, D facilitate the inversion of the friction matrix:
A = (ω2ξ12)/(Mw2ω1) + (ξ13(1 − ω2))/(Mw3ω1)
B = ξ12/Mw2 − ξ13/Mw3
C = (ω1ξ12)/(Mw1ω2) + (ξ23(1 − ω1))/(Mw3ω2)
D = ξ12/Mw1 − ξ23/Mw3

 

- Reciprocal Scaling Term (BDCA.m)
CF12G12 = (ω1ω2Mw1Mw2Mw33)/(Mw1Mw3ξ12ξ23ω2 + Mw2Mw3ξ13ξ12ω1 + Mw1Mw2ξ13ξ23ω3)

 

- Final Transport Coefficients (COEFFICIENTS.m)
The final Fickian-like coefficients F1, F2, G1, G2 are, where σij = (∂μi)/(∂ωj) (computed in DMiOi.m):
F1 = CF12G12 · (C σ11 + B σ21)

F2 = CF12G12 · (D σ11 + A σ21)

G1 = CF12G12 · (C σ12 + B σ22)

G2 = CF12G12 · (D σ12 + A σ22)

 

PDE & Velocity Execution (MainGUFDM.m & VELOCITY.m)

 

- Reference Velocity
The center-of-mass velocity $v$ ensures volumetric consistency, where Ji' is the bracketed part of the flux Ji = -ρ Ji'.
v = ((ρ01 − ρ03) / (ρ01ρ03)) J1'
+ ((ρ02 − ρ03) / (ρ02ρ03)) J2'

 

- Time Evolution
Components 1 and 2 evolve according to:
ωit+Δt = ωit − ( (1/ρ)(∂Ji/∂z) + v(∂ωi/∂z) ) Δt

 

- Spatial Discretization MESHS.m
The spatial derivatives are discretized using the GUFDM non-uniform grid.
